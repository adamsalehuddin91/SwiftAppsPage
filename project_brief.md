## 3. Ekosistem SwiftApps (Active Portfolio)
Sistem kritikal yang dibangunkan dan diuruskan dengan standard produksi tinggi:

* **LorryTech OS (Fleet Management):** Centralized dashboard untuk pemilik lori SME. Menguruskan revenue (Lalamove + side jobs), *digital receipt vault* (LHDN 7-year compliance), komisyen pemandu, dan analitik P&L.

    * *Stack: Next.js 16, React 19, TypeScript, Supabase (PostgreSQL).*
* **TerasEdu:** Solusi pengurusan tadika digital untuk automasi operasi harian guru dan pentadbir.
* **BusinessSystem Management:** Sistem pengurusan kontraktor/pembinaan (Invoicing, PO, Inventory, Finance).
    * *Stack: Laravel 12, React 18, Inertia.js, Supabase PostgreSQL.*
* **Stock Monitor Bursa:** Dashboard pemantauan saham Bursa Malaysia dengan sokongan PWA (Offline-first) dan visualisasi data real-time.
    * *Stack: React 19, Vite 6, Tailwind CSS, Chart.js.*

## 4. Senior-Level Software Architecture
Sistem anda tidak sekadar berfungsi, ia dibina dengan standard industri paling terkini:

### **Core Engineering & Patterns**
* **Bleeding-Edge Frameworks:** Pakar dalam **Laravel 12 (Modern Monolith)** dan **Next.js 16 (Decoupled)** untuk fleksibiliti mengikut keperluan projek.
* **Modern React Mastery:** Penggunaan **React 19** dan **Server Components** untuk prestasi UI yang luar biasa laju.
* **Architectural Patterns:** Implementasi **Service & Repository Pattern** untuk memisahkan *business logic* daripada controller — memastikan kod mudah diuji dan di-skala.
* **SaaS & Multi-Tenancy:** Rekabentuk pangkalan data yang menyokong model perniagaan SaaS dari hari pertama.

### **Data & Infrastructure Engineering**
* **Database Orchestration:** Pengoptimuman **PostgreSQL** melalui **Supabase (Transaction Mode/Pooling)** untuk integriti data tinggi.
* **DevOps Automation:** Strategi **Self-hosted** menggunakan **Hetzner VPS** melalui **Coolify (PaaS)** untuk kawalan penuh infrastruktur dan kos yang efektif.
* **Compliance Ready:** Arsitektur yang mematuhi keperluan audit digital (LHDN compliance) untuk sistem kewangan dan logistik.
* **PWA & Real-time:** Membangunkan aplikasi yang boleh di-install (PWA) dengan visualisasi data menggunakan **Chart.js** dan **Vite 7**.
